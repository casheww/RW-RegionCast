RegionCast

WIP - features listed on this page are not yet available.

A BepInEx mod for Rain World that adds Discord Rich Presence.
This site lists the modded regions compatible with RegionCast.

How does it work?
An owner of the "application" registered on Discord must upload the
thumbnail of the custom region to Discord's Rich Presence system.
This means that if a region creator wants to have their thumbnail art
featured, they must request it themselves.

The mod itself has to know which regions have thumbnails available
for use, which is the purpose of this site. I have a little jQuery script
loading in the custom region data from a JSON, which must be kept up to date.
Discord could provide more useful request responses... but they don't, so here we are.
